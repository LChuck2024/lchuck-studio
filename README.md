# LChuck Studio - Personal Portfolio

个人作品集与创作空间，展示 Agent 集合、App 集合和博客文章。

## 本地运行

**前置要求:** Node.js

1. 安装依赖:
   ```bash
   npm install
   ```

2. 启动开发服务器:
   ```bash
   npm run dev
   ```

3. 在浏览器中打开 http://localhost:3000

## 构建生产版本

```bash
npm run build
```

## 预览生产构建

```bash
npm run preview
```
